# תיאור האפליקציה ל-App Store

## שם
Navigate

## כותרת משנה (Subtitle)
ניהול ניווטים צבאיים

## תיאור

Navigate היא אפליקציה לניהול וביצוע ניווטים צבאיים — מההכנה ועד התחקור.

האפליקציה מאפשרת למפקדים לתכנן ניווטים, להגדיר מסלולים ונקודות ציון, לעקוב אחרי מנווטים בזמן אמת ולתחקר ביצועים. מנווטים מקבלים את המסלול, מבצעים את הניווט ומדווחים על הגעה לנקודות.

תכונות עיקריות:

- תכנון ניווט מלא: הגדרת שטחות, נקודות ציון, גבולות גזרה ונקודות בטיחות
- חלוקת מסלולים אוטומטית או ידנית למנווטים
- מעקב GPS בזמן אמת עם תצוגת מפה
- שלב למידה: המנווטים לומדים את המסלול לפני היציאה
- בדיקת מערכת: אימות GPS, סוללה וקליטה לפני תחילת הניווט
- ווקי טוקי: תקשורת קולית בין מפקדים ומנווטים
- חישוב ציונים אוטומטי
- תחקור מפורט עם מפת מסלול ונתוני ביצוע
- ייצוא דוחות ל-PDF, Excel, GPX ו-KML
- עבודה אופליין מלאה עם סנכרון אוטומטי
- ניהול יחידות, משתתפים והרשאות

האפליקציה תומכת בעברית ומותאמת לשימוש בשטח.
